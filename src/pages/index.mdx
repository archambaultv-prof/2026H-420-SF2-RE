# Structures de données et POO

Bienvenue sur le site du cours Structures de données et POO (420-SF2-RE) du [Collège de
Maisonneuve](https://www.cmaisonneuve.qc.ca/). Ce site présente le contenu du cours dispensé
par le professeur [Vincent Archambault-Bouffard](https://github.com/archambaultv-prof) durant la session d'hiver 2026.

## Organisation du site

Le site est organisé en deux sections principales :

- **Échéancier** : Le calendrier détaillé du cours, semaine par semaine
- **Évaluations** : Les informations relatives aux évaluations du cours

Toutes les sections sont accessibles via le menu de navigation situé en haut de la page.

## Communication

Les étudiants du cours peuvent me contacter via la messagerie MIO dans
[Omnivox](https://cmaisonneuve.omnivox.ca/). Les personnes externes au
Collège de Maisonneuve peuvent me joindre par
[courriel](mailto:varchambaultbouffard@cmaisonneuve.qc.ca).

## Plateforme Omnivox

Ce site web complète mais ne remplace pas
[Omnivox](https://cmaisonneuve.omnivox.ca/). Les étudiants doivent continuer à
consulter régulièrement Omnivox pour les annonces officielles et les informations administratives.

De plus, en cas de divergence entre les informations présentées sur ce site et
celles du plan de cours, le plan de cours prévaut.
