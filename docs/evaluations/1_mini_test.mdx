# Mini test

- **Pondération** : 5%
- **Date** : Mercredi le 11 février 2026
- **Durée** : 45 minutes
- **Matériel autorisé** : Aucun
- **Lieu** : Salle de classe habituelle
- **Sur papier, aucun ordinateur autorisé**

Le mini test consiste à écrire un court programme Python, similaire à
l'exercice [Un terminal dont vous êtes le
héros](https://archambaultv-prof.github.io/programmation-python/docs/basic/stuctures_controles/Exercices/jeu_de_role).

L'objectif du mini test est d'évaluer votre niveau en programmation Python et
votre maîtrise des concepts vus dans le cours SF1. La note du mini test sera
simplement l'une des trois notes suivantes :

- 5/5, vous avez démontré une bonne maîtrise des concepts
- 3/5, vous avez démontré une maîtrise partielle des concepts
- 1/5, vous n'avez pas démontré une maîtrise suffisante des concepts
